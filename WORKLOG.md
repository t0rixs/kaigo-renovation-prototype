## 2026-07-21 - Android実機への最新版インストール

### 目的

- 接続中のスマートフォンへ、現在の`main`最新版をインストールして起動できる状態にする。

### 実施内容

- `flutter devices`でAndroid 14の実機`CPH2557`がUSB接続されていることを確認した。
- `flutter install --debug -d AUBILBMVYH4LIJDQ`でデバッグ版APKをインストールした。
- ADBからアプリの`MainActivity`を明示的に起動した。
- `flutter install`が旧版を一度アンインストールする動作だったため、今後はAndroidのアプリ内データを維持する`adb install -r -t`方式を使うよう`AGENTS.md`を修正した。

### デバッグ・確認

- `build/app/outputs/flutter-apk/app-debug.apk`のインストールが終了コード0で完了した。
- ADBの起動結果が`Status: ok`、`LaunchState: COLD`になったことを確認した。
- 起動後にアプリのプロセスIDを取得でき、端末上で稼働していることを確認した。

### 結果

- Android実機`CPH2557`へ最新版をインストールし、アプリを起動した。
- 今回の更新では旧版のアンインストールが実行されたため、以前のアプリ内ローカルデータは消去されている可能性がある。
- 次回以降は、既存データを維持する上書きインストール方式を使用する。
