# 介護住宅改修 図面・見積プロトタイプ

介護住宅改修向けの図面作成・簡易見積プロトタイプです。

## 主な機能

- 基本情報ページ
- 品番ページ
- 図面ページ
- 見積もりページ
- 1マス = 100mm の方眼図面
- 部屋・設備・施工線の配置
- 図面の拡大・縮小
- 部屋サイズの数値入力
- 設備記号の回転
- 手すり・スロープ・段差解消の簡易見積反映

## 起動方法

ローカルで確認する場合:

```bash
python3 -m http.server 4173
```

ブラウザで以下を開きます。

```text
http://127.0.0.1:4173/
```
